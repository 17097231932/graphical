安装 Graphical

这部分文档包含了 Graphical 的安装过程，使用任何软件的第一步就是正确地安装它。

pip install graphical-formula

要安装 Graphical，只要在你的终端中运行这个简单命令即可：

pip install graphical-formula

如果你没有安装 pip , 可以通过源码安装

获得源码

Graphical 一直在 Github 上积极地开发，你可以一直从这里获取到代码。 你可以克隆公共版本库：

git clone git://github.com/17097231932/graphical.git

获得代码之后，你就可以轻松的将它嵌入到你的 python 包里，或者安装到你的 site-packages:

cd graphical & pip install .

使用内置公式

在 Graphical 中使用公式非常简单

一开始要导入 Graphical 模块：

import graphical

注意: 它在Python中导入模块名为 graphical

然后, 尝试计算某个图形公式。本例子中，我们来获取正方体的面积

figure = graphical.square_area(a=1)

现在，我们有一个名为 area 的 square_area 对象， 我们可以从这个对象中获取所有我们想要的信息。

Graphical 简便的 API 意味着所有 HTTP 请求类型都是显而易见的。

例如，你可以这样获取正方体的周长：

figure = graphical.square_perimeter(a=1)

漂亮，对吧？那么其他图形公式：长方形面积 以及 长方形周长 又是如何的呢？都是一样的简单：

figure = graphical.rectangle_area(a=1, b=2)

figure = graphical.rectangle_perimeter(a=1, b=2)

都很不错吧，但这也仅是 Graphical 的冰山一角呢。

创建并使用自定义公式

Graphical 允许你自定义公式，这非常简单。

第一步，定义一个继承 graphical.Graphical 的类

class MyFigure(graphical.Graphical):

第二步，添加formula属性，值为公式字符串

    formula = 'a+a'

第三步，添加args属性，值为描述这个公式的参数的列表， parameter 函数可以帮我们快速生成它

    args = parameter('a')

好了，我们定义了一个公式，这个公式是 a+a ,参数是 a 。现在让我们调用它吧！

figure = MyFigure(a=1)

打印他，结果是2，一个简单的公式就定义好了，多简单啊！
